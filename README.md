# YouTube Video Downloader 🎥

A simple and efficient Python script to download YouTube videos in the best available quality using the `yt-dlp` library. 

## 🚀 Features
* Downloads videos in the highest resolution automatically.
* Saves the video with its original YouTube title.
* Easy to read and beginner-friendly codebase.

## 🛠️ Prerequisites
Before running the script, make sure you have Python installed on your system. You will also need to install the required library:

```bash
pip install yt-dlp
```

## 💻 How to Use
Clone this repository or download the video_indirici.py file.

Open the file in your favorite text editor (like VS Code or IDLE).

Replace the video_linki variable with the URL of the YouTube video you want to download.

Run the script: ``` python video_indirici.py ```
The video will be downloaded to the exact same folder where your script is located.
